# Public skills

This directory contains Skills accepted through [the promotion policy](../docs/promotion-policy.md).

- [`doc-steward`](doc-steward) — audit and standardize agent-facing repository documentation with a read-only default and gated low-risk apply path.

A promoted skill may later remain with lifecycle `deprecated` during its migration window, but default collections exclude it. Do not create placeholders to make an installer or plugin manifest appear functional.
